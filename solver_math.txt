================================================================================
  PART 1 — HIGH-LEVEL OVERVIEW (for any engineer)
================================================================================

What this solver does:
  Given a 2D cross-section of a body (metal, coated metal, dielectric),
  compute how much radar energy it scatters back toward the transmitter.
  The output is the Radar Cross Section (RCS) in dBsm as a function of
  frequency and incidence angle.

How it works (plain English):
  1. The body's surface is divided into small straight-line panels.
  2. On each panel, we place an unknown "current" (a density function).
  3. We write an equation that says the total field (incident + scattered)
     must satisfy the boundary condition on the surface (zero tangential
     E-field for PEC, impedance condition for IBC, continuity for dielectric).
  4. This gives a linear system: [Matrix] × {currents} = {excitation}.
  5. We solve the linear system to find the currents.
  6. We project the currents to the far field to get the scattered amplitude.
  7. RCS = |amplitude|² / (4k), normalized to a line source.

Why boundary integrals (not FEM or FDTD):
  The 2D Helmholtz equation has an exact Green's function — the Hankel
  function H₀⁽²⁾(kr). This means we only need unknowns ON THE SURFACE,
  not throughout the volume. For a body with perimeter P discretized into
  N panels, the unknowns scale as O(N) instead of O(N²) for volume methods.
  The tradeoff: the matrix is dense (every panel interacts with every other),
  giving O(N²) assembly and O(N³) solve — but N is the perimeter count,
  which is typically 100-2000 for 2D bodies at GHz frequencies.

What the FMM does:
  The Fast Multipole Method replaces the O(N²) dense matrix with an O(N log N)
  matrix-vector product by grouping distant panels into multipole expansions.
  Nearby panels are still computed exactly. This lets the solver use GMRES
  (an iterative solver) without ever forming the full matrix. FMM becomes
  faster than dense at roughly 1000+ panels.

Polarization:
  In 2D, electromagnetic waves split into two independent polarizations:
  - TE (transverse electric): E-field is out of the plane (Ez scalar).
    The unknown is the surface current density (jump in dH/dn).
  - TM (transverse magnetic): H-field is out of the plane (Hz scalar).
    The unknown is the SLP density (related to jump in H across the surface).

Materials supported:
  - PEC (perfect electric conductor): zero tangential E-field.
  - IBC (impedance boundary condition): Et = Zs × (n × H).
  - Homogeneous dielectric: continuous tangential E and H across interface.
  - Multi-layer coatings: arbitrary stacking of dielectric layers over PEC.
  - Lossy magnetic materials: complex ε and μ with both electric and
    magnetic loss tangents.


================================================================================
  PART 2 — THE GREEN'S FUNCTION AND FUNDAMENTAL INTEGRAL EQUATIONS
================================================================================

2.1  The 2D Helmholtz Equation
------------------------------
  All fields satisfy (∇² + k²)u = 0 in each homogeneous region, where
  k = ω√(εμ) is the wavenumber. For lossy materials, k is complex.

2.2  The Free-Space Green's Function
-------------------------------------
  The fundamental solution to (∇² + k²)G = -δ(r - r') in 2D is:

      G(r, r') = (j/4) H₀⁽²⁾(k|r - r'|)

  where H₀⁽²⁾ is the Hankel function of the second kind, order zero.
  This represents a cylindrical wave radiating outward from r'.

  Physical meaning: G is the field at r due to a unit line source at r'.
  The (j/4) prefactor and H₀⁽²⁾ convention ensure outgoing-wave behavior
  (Sommerfeld radiation condition) at infinity.

  Implementation: scipy.special.hankel2(0, kr).
  For real k, H₀⁽²⁾(x) = J₀(x) - jY₀(x) where J₀, Y₀ are Bessel functions.
  The C extension (fmm_near.c) uses the C math library's j0()/y0() for speed.

2.3  Layer Potentials
---------------------
  Given a density σ on a boundary Γ, define:

  Single-Layer Potential (SLP):
      (Sσ)(r) = ∫_Γ G(r, r') σ(r') ds(r')

  This creates a field that is continuous across Γ but whose normal
  derivative has a jump:
      ∂(Sσ)/∂n|₊ - ∂(Sσ)/∂n|₋ = -σ

  Double-Layer Potential (DLP):
      (Kσ)(r) = ∫_Γ ∂G(r, r')/∂n(r') σ(r') ds(r')

  This creates a field whose value has a jump across Γ:
      (Kσ)|₊ - (Kσ)|₋ = -σ

  The jumps are the Plemelj-Sokhotski formulas, and they appear as
  ±½M terms (mass matrix) in the discretized equations.

2.4  Normal Derivative Kernels
------------------------------
  The normal derivative of G with respect to the source normal n' (for K):
      ∂G/∂n' = (jk/4) H₁⁽²⁾(kr) (n' · (r-r'))/|r-r'|

  The normal derivative with respect to the observer normal n (for K'):
      ∂G/∂n = -(jk/4) H₁⁽²⁾(kr) (n · (r-r'))/|r-r'|

  K' is the adjoint of K (transpose of the kernel). The MFIE uses K',
  the coupled formulation uses K.

  Implementation: _dgreen_dn_src() and _dgreen_dn_obs() in rcs_solver.py.
  Uses scipy.special.hankel2(1, kr) for H₁⁽²⁾.


================================================================================
  PART 3 — DISCRETIZATION (GALERKIN METHOD OF MOMENTS)
================================================================================

3.1  Linear Elements
--------------------
  The boundary Γ is divided into N straight-line panels. On each panel,
  the density is represented by linear basis functions:

      σ(r(t)) = (1-t) σ₀ + t σ₁     for t ∈ [0, 1]

  where σ₀, σ₁ are the nodal values at the panel endpoints.
  Adjacent panels share endpoint nodes, giving C⁰ continuity.

  Total DOFs = number of unique nodes (≈ N for closed contours, N+1 for open).

3.2  Galerkin Testing
---------------------
  The integral equation is tested (weighted) with the same linear basis
  functions. For the SLP operator, the (i,j) matrix entry is:

      S[a,b] = ∫₀¹ ∫₀¹ φₐ(tₒ) G(r(tₒ), r'(tₛ)) φ_b(tₛ) Lₒ Lₛ dtₒ dtₛ

  where φₐ, φ_b are the linear shape functions, Lₒ, Lₛ are the panel
  lengths, and r(t) = p₀ + t(p₁ - p₀) maps the reference interval to
  physical space.

  This double integral is computed by tensor-product Gauss-Legendre
  quadrature with 8 points per dimension (64 kernel evaluations per
  panel pair). The quadrature order adapts to 16 for close panels.

3.3  Singular and Near-Singular Integration
-------------------------------------------
  Self-interactions (source panel = observer panel):
    The kernel G has a logarithmic singularity at r = r'.
    The solver uses recursive subdivision with analytic extraction of the
    log(|t-s|) singularity, integrating the smooth remainder numerically.

  Touching panels (shared node):
    The kernel is finite but varies rapidly near the shared vertex.
    Duffy-type transforms map the near-singularity to a smooth integrand
    that standard Gauss quadrature can handle.

  Close but non-touching panels:
    Adaptive quadrature order (5-16 points) based on the distance/size
    ratio. The function _near_singular_scheme() determines the order.

  Implementation: _sk_blocks_near_linear() dispatches to the appropriate
  quadrature path. The C extension (fmm_near.c) handles the regular
  (non-self, non-touching) case with variable-order Gauss quadrature
  for real wavenumbers.

3.4  Mass Matrix
----------------
  The ±½ jump terms from the Plemelj formulas produce a mass matrix:

      M[a,b] = ∫₀¹ φₐ(t) φ_b(t) L dt

  For linear elements on a single panel, this is the standard 2×2 mass
  matrix: L/6 × [[2,1],[1,2]]. It is assembled element-by-element.


================================================================================
  PART 4 — FORMULATIONS (which equations are solved)
================================================================================

4.1  TM PEC — Robin MFIE (Magnetic Field Integral Equation)
------------------------------------------------------------
  Representation: u_scat = S[σ] (single-layer potential)
  Boundary condition: ∂u/∂n = 0 on PEC (TM: n × E = 0 → ∂Hz/∂n = 0)

  Taking the exterior limit of the normal derivative:
      (-½M + K')σ = -∂u_inc/∂n

  This is the MFIE. It uses the K' operator (normal derivative at observer).

  For IBC surfaces, the Robin condition ∂u/∂n + αu = 0 gives:
      (-½M + K' + αS)σ = -(∂u_inc/∂n + α u_inc)

  where α = jk₀Zₛ/Z₀ (TM) encodes the surface impedance.

  Implementation: _solve_tm_robin_mfie() in rcs_solver.py.
  FMM-accelerated path: builds FMMOperator for S and K', solves with GMRES.

  Why MFIE for TM: The MFIE is a Fredholm equation of the second kind
  (the ½M term makes the operator = identity + compact), which gives
  good conditioning and guaranteed unique solutions away from interior
  resonances.

4.2  TE PEC — EFIE (Electric Field Integral Equation)
------------------------------------------------------
  Representation: u_scat = S[σ]
  Boundary condition: u = 0 on PEC (TE: Ez = 0)

  Taking the exterior limit of the potential:
      Sσ = -u_inc

  This is the EFIE. It is a Fredholm equation of the first kind (no ½M
  term), so it is less well-conditioned than the MFIE.

  Implementation: part of the coupled trace formulation in
  _build_coupled_matrix_linear(). The CFIE (Combined Field IE) option
  adds α times the MFIE to cure interior resonances:
      (1-α)·EFIE + α·MFIE

4.3  Coupled Trace Formulation (TE PEC with CFIE)
---------------------------------------------------
  For TE PEC, the solver uses a coupled system with two unknowns per node:
  u (trace = boundary value) and q (flux = normal derivative):

      [½M + K,  -S ] [u]   [u_inc    ]
      [D,    ½M+K' ] [q] = [∂u_inc/∂n]

  where D is the hypersingular operator (computed via the Maue identity
  from the SLP block to avoid hypersingular quadrature):

      D_ij = -k²(n·n')S_ij + (1/LₒLₛ)[[1,-1],[-1,1]] Σ S_ij

  This 2N×2N system is well-conditioned and handles arbitrary PEC/IBC
  combinations for TE.

4.4  Dielectric Bodies — Indirect Two-Density Formulation
-----------------------------------------------------------
  For a single dielectric body (air/dielectric interface), two densities
  are placed on the interface:

      u_scat = K[μ] - S₁[τ]    (exterior, wavenumber k₀)
      u_int  = K[μ] - S₁[τ]    (interior, wavenumber k₁)

  Continuity of u and ∂u/∂n across the interface gives a 2N×2N system.
  The wavenumber k₁ = k₀√(ε_r μ_r) appears in the interior operators.

  Implementation: _solve_dielectric_indirect().

4.5  Multi-Region Indirect SLP Formulation (Layered Coatings)
---------------------------------------------------------------
  For bodies with multiple material layers (e.g., PEC core + absorber +
  radome), each interface gets its own SLP density:

  Interface types:
    - Air/dielectric (TYPE 3): transmission condition (u and ∂u/∂n continuous)
    - Dielectric/PEC (TYPE 4): PEC or Robin BC
    - Dielectric/dielectric (TYPE 5): transmission between two dielectrics

  The system is assembled interface-by-interface:

  For a transmission interface between regions m and p:
    Flux row: (-½M + K'_m)σ_m - β⁻¹(½M + K'_p)τ_p = -RHS
    Trace row: S_m σ_m - S_p τ_p = -RHS

  where β = μ_p/μ_m (TE) or ε_p/ε_m (TM) is the material contrast ratio,
  and the operators S_m, K'_m use wavenumber k_m of region m.

  Cross-coupling: if two interfaces share a region, their densities couple
  through that region's Green's function. The operator S_m(src_on_iface_j)
  evaluated at observer nodes on iface_i provides this coupling.

  For a PEC/Robin interface:
    (+½M + K' + αS)σ = -RHS  (density on the dielectric side only)

  Implementation: _solve_multi_region_indirect() in rcs_solver.py.
  FMM path: builds FMM operators per (wavenumber, source_mask) pair,
  constructs block matvec, solves with preconditioned GMRES.

4.6  Far-Field Projection
--------------------------
  Once the surface density σ is known, the far-field scattered amplitude
  in direction θ is:

      A(θ) = ∫_Γ σ(r') exp(jk d̂·r') ds(r')

  where d̂ = (cos θ, sin θ) is the observation direction.

  The monostatic RCS (backscatter) uses θ = incidence angle.
  The 2D RCS is:

      σ_2D = |A|² / (4k)

  in units of meters (or the geometry units). Reported in dB: 10 log₁₀(σ_2D).


================================================================================
  PART 5 — FAST MULTIPOLE METHOD (FMM)
================================================================================

5.1  Core Idea
--------------
  The dense matrix-vector product y = A·x requires N² kernel evaluations.
  Most pairs (i,j) are far apart, and the Green's function between distant
  groups can be represented by a compact multipole expansion.

  The FMM replaces the N² direct evaluations with:
    - Near field: direct computation for close panel pairs (O(N) total)
    - Far field: multipole expansions + translations (O(N log N) total)

5.2  The Addition Theorem
--------------------------
  The 2D Helmholtz Green's function satisfies:

      H₀⁽²⁾(k|r-r'|) = Σ_{n=-p}^{p} J_n(k|r'-c|) H_n⁽²⁾(k|r-c|) e^{in(θ-θ')}

  where c is an expansion center, and (|r-c|, θ), (|r'-c|, θ') are polar
  coordinates relative to c. This separates the source r' from the
  observer r through the center c.

  J_n: Bessel function of the first kind (regular, bounded at origin)
  H_n⁽²⁾: Hankel function of the second kind (outgoing, singular at origin)

  Truncation at order p gives exponentially decaying error. The solver
  uses p = ceil(k·D) + n_digits + 2, where D is the box diameter and
  n_digits controls accuracy (default 6 → ~10⁻⁶ relative error).

5.3  Tree Structure
-------------------
  An adaptive quadtree subdivides the panel centers until each leaf has
  ≤ max_leaf panels (default 20-40). For elongated geometries (like the
  fin_bumper), this produces 5-8 levels with 50-100 leaves.

  A completeness check ensures every leaf pair is covered: either in the
  near list (direct computation) or through the interaction list at some
  ancestor level (multipole translation). This prevents partition gaps
  that would cause accuracy loss on elongated geometries.

5.4  Translation Operators
---------------------------
  P2M (Particle to Multipole): at each leaf, aggregate source panel
  contributions into outgoing multipole coefficients:
      O_n = Σ_sources σ_j J_n(k|r_j-c|) e^{-inθ_j}

  M2M (Multipole to Multipole): shift child's outgoing expansion to
  parent center using Bessel J addition theorem.

  M2L (Multipole to Local): convert a distant source group's outgoing
  expansion into a local (regular) expansion at the observer group center,
  using Hankel H addition theorem.

  L2L (Local to Local): shift parent's local expansion to child center
  using Bessel J addition theorem.

  L2P (Local to Particle): evaluate the local expansion at each observer
  quadrature point. For the SLP, this gives the field value. For K',
  this gives the normal derivative (requires gradient of J_n).

  All translations are implemented as matrix-vector products using
  vectorized scipy Bessel evaluations. The M2M/M2L/L2L matrices are
  (2p+1) × (2p+1) where p is the truncation order.

5.5  Near-Field Caching
------------------------
  Near-field interactions (self, touching, and close panel pairs) are
  assembled into a dense matrix at FMM build time. This matrix is
  multiplied with the density vector at each GMRES iteration — a fast
  BLAS operation. The assembly uses:
    - C extension (fmm_near.c) for real wavenumber with libm Bessel functions
    - Vectorized scipy Hankel evaluation for complex wavenumber
    - Python recursive quadrature for self and touching pairs

  The near field typically covers 15-30% of all panel pairs.

5.6  FMM in the Multi-Region Solver
------------------------------------
  For multi-material geometries, each interface at each wavenumber needs
  its own FMM operator (S and K'). A shared quadtree is built once and
  reused across all operators. Source element masks restrict which panels
  contribute to each operator.

  The block system Asys·x = b is solved with preconditioned GMRES, where
  the preconditioner is the block-diagonal extracted from the near-field
  matrices (which capture the dominant self-interaction for each interface).

================================================================================
  PART 7 — VALIDATION AND ACCURACY
================================================================================

  The solver is validated against the Mie series (exact solution for
  circular cylinders). Validated cases:

  PEC:
    TE ka=1:  0.002 dB error (80 panels)
    TM ka=1:  0.001 dB error (80 panels)

  Dielectric (εr=4):
    TE ka=1:  0.007 dB error
    TM ka=1:  0.059 dB error

  IBC (Zs=1+j):
    TE ka=1:  0.003 dB error
    TM ka=1:  0.002 dB error

  Multi-layer (3-layer coating):
    Degenerate test (3 identical layers = single layer): 0.006 dB error
    Residual: machine precision (10⁻¹⁵)

  FMM accuracy:
    SLP matvec vs dense:  3.6×10⁻¹⁶ to 5.5×10⁻⁷ (depends on mesh size)
    Full RCS vs Mie:      0.000-0.002 dB (with FMM+GMRES)
    Complex-k operators:  1.2×10⁻⁸ relative error

  The mesh discretization (λ/20 panel size) is always the dominant error
  source, not the BIE quadrature or FMM truncation.
